# zinnia-demo

## Requirement
- ruby 2.2.0
- rails 4.1.7
- zinnia-0.0.6
- zinnia-tomoe-0.6.0

## Zinnia

### Download
https://sourceforge.net/projects/zinnia/files/

### Installation

```
cd zinnia-0.0.6/ruby
ruby extconf.rb
make
make install
```

```
cd zinnia-tomoe-0.6.0-20080911
./configure
make
make install
```

## APIs
- home/top (GET)
- home/classify (GET)
